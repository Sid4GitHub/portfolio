## Personal Projects

### PcRemote (2017)
- **IoT / Man-Machine Interaction**: Microcontroller receives IR data from a remote and sends it to the computer.
- **Tech**: Java used for receiving codes and performing operations.

### Home Automation System (2016)
- **IoT Solution**: Complete electronics and software solution for room automation.
- **Control**: Operated by IR remote and custom Android app via Bluetooth/Internet.
- **Features**: Control lights, alarms, and RGB LED strips; display date/time/temperature.

### Virtual Mouse (2015)
- **Computer Vision**: Java-based application simulating a mouse.
- **Mechanism**: Processes webcam images to detect objects and perform mouse operations based on coordinates.

### Features Extraction from MRI (2014)
- **Image Processing**: Java-based application processing MRI images.
- **Output**: Returns binary images of extracted White Matter, Gray Matter, CSF, Skin, and Fat.

### Steganography (2013)
- **Security**: Java-based application for data security with encryption and decryption.
- **Method**: Uses image files as cover media to encode/decode secret data.
